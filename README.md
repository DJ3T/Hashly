# Hashly  

<img src="./logo.png" alt="Logo Hashly" width="10%" />

Hashly is a college cybersecurity project that automatically detects the hashing algorithm of a given hash, suggests the best cracking method, and generates ready-to-use commands for the tool Hashcat.

Hashly est un projet universitaire qui détecte automatiquement l'agorithme de hashing d'un hash donné, qui propose une méthode adaptée de craking et génère une commande prête à l'emploi pour l'outil Hashcat.
